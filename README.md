
# API endpoints

## addition

Calculates result of a+b

    http://localhost:7777/api/v1/add/<a>/to/<b>

## subtraction

Calculates result of b-a

    http://localhost:7777/api/v1/subtract/<a>/from/<b>


## multiplication

Calculates result of a*b

    http://localhost:7777/api/v1/multiply/<a>/by/<b>

## division

Calculates result of a/b

    http://localhost:7777/api/v1/multiply/<a>/by/<b>
